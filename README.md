# Data_Analytics
This project presents an end-to-end Retail Sales Analysis solution that combines Python-based data preprocessing with an interactive Power BI dashboard for business insights. The goal of this project is to analyze sales performance, identify key trends, and visualize metrics that help in data-driven decision-making for retail businesses.

Key Features:

Data Cleaning & Processing:
Used Python (Pandas, NumPy, Matplotlib) to clean, transform, and analyze raw sales data.

Exploratory Data Analysis (EDA):
Derived insights on top-selling products, customer demographics, profit margins, and regional sales trends.

Interactive Power BI Dashboard:
Designed a dynamic dashboard with filters for product categories, time period, and region to help visualize KPIs such as:

Total Sales

Profit by Product & Region

Year-over-Year Growth

Customer Purchase Patterns

Business Insights:
Helped identify underperforming segments and high-profit categories, supporting strategic marketing and inventory decisions.
